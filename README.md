# n8n AI Customer Support Agent

This repository contains an AI-powered customer support automation project built with n8n, Telegram, OpenAI, and Google Sheets.

The project files are organized in this repository with exported n8n workflows, documentation, and screenshots.

## Features

- Telegram customer support bot
- AI-powered FAQ replies
- Human support escalation
- Google Sheets logging
- Admin notification via Telegram

## Tech Stack

- n8n
- Telegram Bot API
- OpenAI
- Google Sheets

## Security

No API keys, bot tokens, or credentials are included in this repository.
All credentials are configured securely inside n8n.
